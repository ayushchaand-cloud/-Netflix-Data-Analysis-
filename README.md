# -Netflix-Data-Analysis-
📊 Netflix Data Analysis using Python and Data Visualization.

This project analyzes Netflix titles dataset using Python, Pandas, Matplotlib, and Seaborn.

## Dataset
- netflix_titles.csv

## Analysis Performed
- Movies vs TV Shows
- Content by Country
- Release Year Trends
- Rating Distribution
- Top Genres

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
